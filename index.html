Works // Selected projects since Aug 2025

Creature Program

Takes inspiration from Lilo and Stitch, using advanced Java methods and variables to bring animated creatures to life. Users interact by clicking a red ball, triggering dynamic passing animations and environmental effects like animated waves.

Memory Mirror

Designed and built an interactive installation that manipulates live video feed via physical controls. Wired an Arduino circuit with potentiometers to dynamically change video pixelation and saturation in a custom Processing sketch, bridging physical user input with digital visualization.

Kallax Smart bin

Integrated ultrasonic and MQ135 air quality sensors with a microcontroller to revolutionize waste management. Programmed the system to analyze bin fill levels and detect pungent odors, transmitting real-time data via LoRa to optimize collection routes.

3D Room Environment

Initial exploration into complex 3D modeling using Blender. Focused on precise composition and realistic textures to create a detailed Japanese-style interior, featuring custom modeled furniture and detailed 'tatami' floor mat textures.

Interactive Tank Model

Modeled a stylized tank in Blender, heavily based on Leonardo Da Vinci's Wooden Tank concept from 1487, then engineered a playable web experience. Implemented controls allowing users to drive the 3D model within a browser environment, showcasing the integration of custom art assets with real-time web game engine logic.
